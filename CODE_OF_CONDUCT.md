# Code of Conduct

## Our Standard

This project is committed to a respectful, professional, and technically rigorous community.

Examples of behavior that contribute to a positive environment include:

- giving actionable, respectful feedback
- assuming good intent while being direct about technical concerns
- sharing context, benchmarks, and reproduction steps
- helping new contributors understand project constraints

Examples of unacceptable behavior include:

- harassment, abuse, or personal attacks
- discriminatory language or conduct
- bad-faith disruption, trolling, or intimidation
- publishing private information without permission

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests, discussions, and other public or private interactions connected to the project.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Reporting

To report unacceptable behavior, open a private security-style report if the platform supports it, or contact the maintainers through GitHub.
