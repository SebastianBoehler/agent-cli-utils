## Maintainer Policy

This project is open to additional maintainers.

Maintainers are expected to:

- keep the project focused on agent-facing CLI tools
- protect stability for downstream automation
- review contributions with low-compute deployment constraints in mind
- help with releases, triage, and community moderation

If you are interested, start by contributing consistently and then open a discussion or issue about maintainership.
